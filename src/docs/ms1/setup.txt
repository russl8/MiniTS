## Running the application

1. Set present working dir to src:
	$ cd src
	
2. To execute the program using a test file, run
	$ java -jar ms1.jar tests/<testFilename>.txt
	
	Ex: java -jar ms1.jar tests/t3.txt
	
3. Refresh the tests folder, and the output html file will be in tests.output
	Ex: src/tests/output/t4output.html
